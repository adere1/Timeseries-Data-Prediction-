Instructions to run the code
1. If node js is not installed please install the node js first using command sudo apt-get install nodejs (sudo doesnt work on remotecs just use apt-get install nodejs)
2. If npm is not installed then install npm using sudo apt-get install npm (not necessary step) (sudo doesnt work on remotecs just use apt-get install npm)
3. Extract the project in any directory use CD command and traverse the directory till you find the "index.js" file 
4. run project using (if using remote cs first run chmod 777 index.js then try ./index.js) ./index.js output will be generated in "output.txt" file 
(works best with local ubuntu or university VM machine might face some installation problems on remote cs because sudo is not available to students)
